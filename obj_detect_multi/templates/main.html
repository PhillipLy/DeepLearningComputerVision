About

This Flask-based web application aims to demonstrate a system that is capable of localizing and identifying multiple objects in the given multimedia source. The core of the system is built upon Googleâ€™s Tensorflow Object Detection API that sits on top of the Tensorflow framework.

This system uses the ssd_inception_v2_coco model provided by Google that has been trained on the COCO dataset. Links to all of the resources used to build this application, including the project's Github repository, can be found on the bottom on the page.
